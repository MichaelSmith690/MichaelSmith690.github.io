Alcohol Delivery


About


Alcohol Delivery is an eCommerce website that offers the users convenience of ordering alcohol whenever they need. Shoppers looking for liquor of various kinds can sign up using their social profile credentials and browse through the website to order from the largest collection of beers, wines, spirits, etc.


What/How did?


Implemented the search functionality of the online marketplace, the wallet integrated with the store, Alcohol delivery system, and the brainstorming to cater. Built a system that provides greater transparency and far more efficient workflows while maintaining complete control over the internal business flows. Designed a brand new digital catalog, implemented a site-wide call to action.

Technology Stack

  • Language: Javascript, TypeScript, HTML, CSS, SQL
  • Framework: React, Node, Express, Nuxt, Redux
  • Database: MongoDB, MySQL
  • Development tools: Git, Gulp
  • Payment Gateway: Reddot Payment
  • Server: AWS Ubuntu NGINX
  • Web Services: Node RESTful API
  • Others: Artisan Queue for task Mail Queue for mailing Hoiio SMS API
